HTML 基础

2021-04-20 3 min read # html

1.网页
构成网站的基本元素，称为HTML文件（由html语言创建）。
2.html
超文本标记语言（Hyper Text Markup Language）,是一种标记语言。
超文本：
1.除了文字还可以加入声音、图片、视频等内容
2.可以从一个文件跳到另一个文件（超级链接文本）

3.常用浏览器
(1).火狐（Firefox） Gecko
(2).谷歌（Chrome） Blink
(3).Safari Webkit
(4).Opera Blink
(5).IE浏览器 Trident
内核作用：读取网页内容，将网页显示出来

4.web标准
为什么需要web标准？ 因为浏览器不同，它们显示的页面或者排版会有一些差异。
web标准构成：
1.结构 对网页元素进行整理和分类 html
2.表现 给网页元素进行设计 css
3.行为 网页模型的定义和交互 javascript
web标准提出的最佳体验方案是结构、样式、行为相分离。
